## 🚀 Deployment

| Platform | Method | Free? |
|---|---|---|
| GitHub Pages | Push to repo → Settings → Pages → Deploy from main | ✅ |
| Netlify | Drag & drop the folder at netlify.com/drop | ✅ |
| Vercel | `vercel deploy` from this folder | ✅ |
